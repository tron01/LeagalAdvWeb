# LeagalAdvWeb



##  First  time only [1-5]

#### 1. Create Virtual Env 
      py -m venv myenv
      py myenv\Scripts\activate

#### 2. Install Packages 
    
    pip install django
    pip install mysqlclient

#### 3. Migrate 
    py manage.py migrate

#### 4. RunServer
    py manage.py runserver

#### To Deactivate Virtual Env 
    py deactivate

## To Run [1-2]

#### 1. Open myenv
    py myenv\Scripts\activate
#### 2. RunServer
    py manage.py runserver
